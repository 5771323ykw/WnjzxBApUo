# Python医药信息管理系统 python849

## 项目概述

Python医药信息管理系统是一款针对药品、顾客及经办人信息进行管理的软件。系统提供信息的查询、录入、删除、修改等功能，并支持数据报表和信息浏览。

## 技术栈

- 前端：HTML、CSS、JavaScript、jQuery
- 后端：Python、Django
- 数据库：MySQL
- 开发工具：PyCharm、Navicat

## 运行环境

- Python 3.0以上
- MySQL 8.0

## 功能模块

- 药品信息管理
  - 信息查询
  - 信息录入
  - 信息删除
  - 信息修改
- 顾客信息管理
  - 信息查询
  - 信息录入
  - 信息删除
  - 信息修改
- 经办人信息管理
  - 信息查询
  - 信息录入
  - 信息删除
  - 信息修改
- 数据报表
- 信息浏览

## 部署步骤

1. 配置Python 3.0以上环境和MySQL 8.0数据库
2. 克隆或下载项目源码
3. 安装所需依赖：`pip install -r requirements.txt`
4. 配置数据库连接信息
5. 运行项目：`python manage.py runserver`
6. 打开浏览器访问对应地址，即可开始使用系统
